# NAATOS TB V1

The NAATOS TB V1 is a modular diagnostic system comprising:

1. Tongue Swab (single-use)
2. Sample Prep Tube (single-use)
3. Sample Prep Module (reusable)
4. Test Consumable (single-use)
5. Power Module (reusable)

The NAATOS Platform encompasses core technologies, components, and interfaces, serving as the foundation for a diverse point-of-care diagnostic product line, each designed for specific disease targets. It ensures an analogous user experience across all products, facilitating seamless interactions. The NAATOS Products are built on this shared foundation, and their common functionalities are deliberately updated driven by user and market needs to enhance the overall user experience as technology advances while upholding compliance, quality, and regulatory regulations.  Starting with addressing the challenges in TB diagnostics opens the door to regional production and innovation, unlocking accessible, quality, and impactful diagnostics.
